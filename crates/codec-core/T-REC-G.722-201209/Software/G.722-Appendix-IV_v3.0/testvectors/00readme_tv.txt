Test vectors to verify correct execution of ITU-T G.722 PLC Appendix IV
Version:      1.1
Release Date: Jul 3, 2007

This directory contains preliminary testvectors to help verification of the execution of the ITU-T G.722 Appendix IV (version 1.1). 

Format: all files contain 16 bit sampled data using little-endian format.

*.bst  - G.192 bit stream file
*.out  - reference output files

Usage: copy decg722.exe in directory "testvectors", run test.bat
