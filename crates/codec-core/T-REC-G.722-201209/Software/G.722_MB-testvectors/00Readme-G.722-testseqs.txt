00readme-G.722-testseqs.txt
===========================

Version 2.0

Read-me file for the digital test sequence distribution for ITU-T G.722 (main body).


This distribution contains the test sequences for G.722, described in detail within G.722 Appendix II. 

Folder g722-ascii contains 17 test sequence files in ASCII format.

Folder g722-ts-be contains 17 test sequence files in binary, big endian format (suitable for CPU architectures such as PowerPC).

Folder g722-ts-le contains 17 test sequence files in binary, little endian format (suitable for CPU architectures such as Intel).
