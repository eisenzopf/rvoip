# 工业级呼叫中心平台升级计划

**Goal:** 将 rvoip web console 从开发者工具升级为工业级呼叫中心管理平台，补全组织架构、号码资源、技能管理、SIP 测试等核心模块。

---

## Phase 7: 基础设施改造 (紧急修复 + 自动化)

### 7A: 坐席管理改造
- 坐席编辑 Dialog (当前只有新增)
- ID 系统自动生成 (AGT-001, AGT-002... 递增)
- SIP URI 基于域名+分机号自动生成 `sip:{ext}@{domain}`
- 分机号从号码池自动分配

### 7B: 部门管理模块 (新)
- 后端: departments 表 + CRUD API
- 前端: 部门管理页面 + 坐席关联部门下拉选择
- 部门层级（支持子部门）

### 7C: 号码资源管理 (新)
- 后端: extensions 表 (分机号池) + number_ranges 表
- 前端: 分机号池管理页 (范围分配、状态标记)
- 分机号分配/回收逻辑

### 7D: 技能组管理 (新)
- 后端: skills 表 + agent_skills 关联表 (含熟练度)
- 前端: 技能定义页面 + 坐席技能分配（多选+等级）
- 替换现有纯文本标签

## Phase 8: SIP 测试与通话体验

### 8A: WebRTC/SIP 软电话 Demo
- 前端内嵌 SIP.js WebRTC 软电话组件
- 坐席可在浏览器直接注册 SIP、接听/拨打电话
- 通话状态实时显示 + 通话控制面板

### 8B: SIP 测试页面
- SIP REGISTER 测试 (注册到 rvoip registrar)
- SIP OPTIONS ping 测试
- 两个浏览器标签互打测试
- 通话质量指标显示 (延迟、丢包)

## Phase 9: 业务流程补全

### 9A: 排班管理
- 班次定义 (早/中/晚)
- 排班日历 (周视图/月视图)
- 签到/签退 + 工时统计

### 9B: IVR 语音导航
- 可视化 IVR 流程编辑器 (拖拽节点)
- 按键导航配置
- 工作时间/非工作时间路由

### 9C: 黑白名单
- 来电黑名单管理
- VIP 白名单
- 号码规则匹配

### 9D: 通话质检
- 质检评分模板 CRUD
- 质检任务分配
- 评分记录
